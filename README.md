# wickra-xray-site

Marketing site for [**wickra-xray**](https://github.com/wickra-lib/wickra-xray) — **xray.wickra.org**.

Built with [VitePress](https://vitepress.dev) and deployed on Cloudflare Pages. The
structure mirrors [webpage](https://github.com/wickra-lib/webpage) (wickra.org) and
[wickra-docs](https://github.com/wickra-lib/wickra-docs) (docs.wickra.org): shared
header, footer, theme, badge pipeline, and workflows.

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # -> .vitepress/dist
```
