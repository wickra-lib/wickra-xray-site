# Privacy

This site — **xray.wickra.org** — is a static marketing and documentation site
for the Wickra X-Ray open-source project. It is built with
[VitePress](https://vitepress.dev) and served as static files.

## What we collect

**Nothing.** This site sets no cookies, runs no analytics or tracking scripts, and
has no backend or database. We do not collect, store, or share any personal data.

## Hosting

The site is served by a static host (Cloudflare Pages). Like any web host, the
provider may process standard server request metadata (such as IP address and
user-agent) transiently to deliver the page and protect against abuse. See the
host's own privacy policy for details. We do not receive or retain this data.

## External links

Pages link out to third-party sites — GitHub, package registries (crates.io, PyPI,
npm, NuGet, Maven Central, pkg.go.dev, r-universe) and the other Wickra sites. Those
destinations have their own privacy policies, which govern your visit once you leave
this site.

## The software itself

Wickra X-Ray, the library, runs entirely on your own machine over data you
provide. It has no telemetry and makes no network calls of its own.
